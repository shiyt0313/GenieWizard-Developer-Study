# ReactGenie

## Introduction

**ReactGenie** is a programming framework that employs an object-oriented state abstraction method to effortlessly support the construction of complex multimodal mobile applications. **ReactGenie** allows developers to define both the content and actions within an app. It automatically manages the natural language understanding and composition of different modalities by generating a parser that utilizes large language models (LLMs).

## Installation

**ReactGenie** can be conveniently installed using NPM.

```bash filename="run in terminal" {3} copy
npm install --save reactgenie-lib
```
